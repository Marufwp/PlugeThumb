Automatically set featured images from YouTube videos or uploaded media library videos with thumbnail generation.

== Description ==

PlugeThumb allows you to automatically set the featured image of your posts from either:

* A YouTube video URL (auto fetches thumbnail)
* A video file uploaded/selected from the WordPress Media Library (sets poster/thumbnail if available)

No more manual screenshot and upload for youtube videos. This plugin saves you time by attaching the right thumbnail whenever you add a video.

= Features =

* Add YouTube URL and auto set featured image
* Select video from Media Library and auto set thumbnail as featured image
* Works with Classic Editor and Gutenberg
* Lightweight and developer friendly
* Supports default WordPress media uploader

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugethumb` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to a post, open the "Video & Featured Thumbnail" meta box (right sidebar).
4. Enter a YouTube URL or select/upload a video from Media Library.
5. Save/Update the post, the featured image will be set automatically.

== Frequently Asked Questions ==

= Does this plugin work with custom post types? =
Yes, any post type that supports featured images can use this plugin.

= What happens if the video has no poster/thumbnail? =
For YouTube videos, the plugin automatically fetches the thumbnail. For uploaded videos, you need to set the Featured Image manually if required.

= Does this plugin support Gutenberg (block editor)? =
Yes, it works with both Gutenberg and Classic Editor.
